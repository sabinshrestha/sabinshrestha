# Customization Checklist

Use this checklist to ensure you've customized all parts of your GitHub profile README.

## ‚úÖ Pre-Setup

- [ ] Create a new repository with the same name as your GitHub username
- [ ] Make the repository **public**
- [ ] Copy the contents of `README.md` from this template

## ‚úÖ Basic Information

### Header & Name
- [ ] Replace `[YOUR NAME]` with your actual name
- [ ] Customize the greeting emoji if desired

### Social Media Links
- [ ] Update `[YOUR_TWITTER_URL]` with your Twitter profile URL
- [ ] Update `[YOUR_TWITTER_HANDLE]` with your Twitter username
- [ ] Update `[YOUR_GITHUB_URL]` with your GitHub profile URL
- [ ] Update `[YOUR_GITHUB_USERNAME]` in all locations (appears 3+ times)
- [ ] Update `[YOUR_LINKEDIN_URL]` with your LinkedIn profile URL
- [ ] Update `[YOUR_LINKEDIN_USERNAME]` with your LinkedIn username
- [ ] Update `[YOUR_WEBSITE_URL]` with your website/blog URL
- [ ] Update `[YOUR_WEBSITE_DOMAIN]` with just the domain name

### Badges
- [ ] Verify all badge URLs are working
- [ ] Remove badges for social media you don't use
- [ ] Test that badges display correctly

## ‚úÖ Content Sections

### Bio
- [ ] Write your bio (2-3 sentences)
- [ ] Mention your primary technologies/skills
- [ ] Add what you're passionate about or working on

### Open Source Contributions
- [ ] Add your actual open source projects
- [ ] Update repository links
- [ ] Write brief descriptions for each
- [ ] OR delete this section if not applicable

### Blog Posts
- [ ] Add your actual blog post titles and links
- [ ] Verify all links work
- [ ] OR delete this section if you don't have a blog

### Trivia Section
- [ ] Update blog link
- [ ] Update Twitter link
- [ ] Update LinkedIn link
- [ ] Add portfolio link
- [ ] Add personal goals or fun facts
- [ ] Customize emojis if desired

## ‚úÖ Professional Skills

### Technology Logos
- [ ] Remove placeholder skill badges
- [ ] Add badges for technologies you actually use
- [ ] Test that all logo images load correctly
- [ ] Organize skills in a logical order
- [ ] Link each badge to the technology's website
- [ ] OR uncomment and customize the example section

### Optional Enhancements
- [ ] Add more technologies
- [ ] Organize by category (Frontend, Backend, DevOps, etc.)
- [ ] Add skill level indicators

## ‚úÖ GitHub Stats

### Basic Stats
- [ ] Replace `[YOUR_GITHUB_USERNAME]` in the stats URL
- [ ] Test that stats display correctly
- [ ] Choose a theme if desired

### Optional Stats Cards
- [ ] Add top languages card
- [ ] Add contribution streak
- [ ] Customize colors and themes

## ‚úÖ Final Touches

### Review
- [ ] Remove all HTML comments if desired

- [ ] Preview the rendered markdown
- [ ] Click all links to verify they work
- [ ] Check on mobile view

### Optional Additions
- [ ] Add custom sections
- [ ] Add animated GIFs or images
- [ ] Add more badges (certifications, achievements)
- [ ] Add "Currently learning" section
- [ ] Add "Looking to collaborate on" section
- [ ] Add custom banners or graphics
