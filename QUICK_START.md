# Quick Start Guide

Get your GitHub profile up and running in 5 minutes!

## âš¡ Fast Setup (5 Minutes)

### Step 1: Copy the Template
Copy the entire contents of `README.md` from this repository.

### Step 2: Create Your Profile Repository
1. Go to GitHub and create a new repository
2. **Important:** Name it exactly the same as your GitHub username (e.g., if your username is `johndoe`, name the repo `johndoe`)
3. Make it **public**
4. Initialize with a README (or replace the default one)

### Step 3: Replace Placeholders

Do a find-and-replace for these key items:

| Placeholder | Replace With | Example |
|-------------|--------------|---------|
| `[YOUR NAME]` | Your actual name | `John Doe` |
| `[YOUR_GITHUB_USERNAME]` | Your GitHub username | `johndoe` |
| `[YOUR_TWITTER_HANDLE]` | Your Twitter username (without @) | `johndoe` |
| `[YOUR_TWITTER_URL]` | Full Twitter profile URL | `https://twitter.com/johndoe` |
| `[YOUR_LINKEDIN_USERNAME]` | Your LinkedIn username | `johndoe` |
| `[YOUR_LINKEDIN_URL]` | Full LinkedIn profile URL | `https://linkedin.com/in/johndoe` |
| `[YOUR_WEBSITE_URL]` | Your website/blog URL | `https://johndoe.com` |
| `[YOUR_WEBSITE_DOMAIN]` | Just the domain name | `johndoe.com` |

### Step 4: Customize Your Bio

Replace this section:
```markdown
> [YOUR BIO - Describe who you are...]
```

With your actual bio (2-3 sentences):
```markdown
> I'm a full-stack developer passionate about building web applications.
> I specialize in React and Node.js and love contributing to open source.
```

### Step 6: Optional Sections

**Don't have a blog?** Delete or comment out the "Popular Blog Posts" section.

**Don't contribute to open source yet?** Delete or comment out the "Open Source Contributions" section.

**Want to keep it minimal?** Just keep the header, bio, skills, and GitHub stats!

### Step 7: Commit and Push

1. Paste your customized content into your profile repository's README.md
2. Commit the changes
3. Push to GitHub
4. Visit your GitHub profile (https://github.com/YOUR_USERNAME) to see it live!

## ðŸ†˜ Troubleshooting

### Profile not showing?
- Make sure your repository name exactly matches your username
- Make sure the repository is **public**
- Make sure the file is named `README.md` (case sensitive)

### Images not loading?
- Check that URLs don't have typos
- Make sure you're using `https://` not `http://`
- Try viewing in an incognito window (cache issue)

### Badges not displaying?
- Verify your username/handles are correct
- Check for special characters that need URL encoding
